# Chess
Game Chess Using Python3 , pygame
Modes:
- 2 players
- Player 1 vs AI minimax 

## Usage
*) To compiler:
```bash
python3 game_chess.py
```
*) To install pygame module:
```bash
sudo apt-get install python3-pygame
```
